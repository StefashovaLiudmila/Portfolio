# Анализ A/B-теста и проверка статистических гипотез

Учебный проект по анализу данных — Яндекс Практикум

---

## О проекте

Проект выполнен в рамках курса «Аналитик данных» (Яндекс Практикум).  
Цель — проверить статистические гипотезы о поведении пользователей и оценить эффективность новой версии интерфейса интернет-магазина.

Проект состоит из двух частей:

1. Проверка гипотезы о различии активности пользователей из Москвы и Санкт-Петербурга  
2. Оценка эффективности A/B-теста новой версии интерфейса

---

## Задачи

### Часть 1. Региональные различия в активности

- Проверить гипотезу о том, что пользователи из Санкт-Петербурга проводят в приложении больше времени, чем пользователи из Москвы
- Использовать тест Манна-Уитни для сравнения двух независимых выборок
- Сформулировать вывод на основе полученного p-value

### Часть 2. Оценка A/B-теста интерфейса

- Проверить корректность проведения теста (баланс групп, пересечения с другими тестами, мощность выборки)
- Оценить влияние новой версии сайта на конверсию в покупку
- Провести статистический тест для долей
- Сформулировать рекомендацию по внедрению

---

## Результаты

### Часть 1. Региональные различия

- Выборки: Москва (6234 пользователя), Санкт-Петербург (2306 пользователей)
- Среднее время активности: Москва — 10.88 ч, Санкт-Петербург — 11.59 ч
- Тест Манна-Уитни (односторонний): p-value = 0.0891 > 0.05
- **Вывод:** статистически значимых различий не обнаружено

### Часть 2. A/B-тест интерфейса

**Проверка корректности теста:**
- Исходная выборка: 10 850 участников
- Выявлены пересечения с конкурирующим тестом: 887 пользователей (8.2%) — исключены
- Итоговая выборка: 9 963 пользователя
- Распределение по группам: A — 49.7% (4 952), B — 50.3% (5 011) — сбалансировано
- Распределение по типам устройств — равномерное
- Расчёт мощности: выборка достаточна для обнаружения эффекта (> 7500 пользователей)

**Результаты теста:**
- Конверсия в группе A (текущая версия): 27.81%
- Конверсия в группе B (новая версия): 29.54%
- Абсолютный прирост: +1.73 п.п.
- Относительный прирост: +6.2%
- z-тест для долей: p-value = 0.0283 < 0.05

**Вывод:** новая версия интерфейса статистически значимо увеличивает конверсию. Рекомендовано внедрение для 100% пользователей.
